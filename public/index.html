MarkItDown REST API

Convert any document to Markdown format

About

This API service converts various file formats to Markdown using Microsoft's MarkItDown library. Simply upload your file and receive clean, formatted Markdown text that's perfect for documentation, content management, or feeding to Large Language Models.

API Endpoints

POST /api/v1/convert

Convert a single file to Markdown format. Supports optional download response.

POST /api/v1/convert/batch

Convert multiple files in a single request.

GET /health

Check service health status.

Supported Formats

PDF Word (DOCX) PowerPoint (PPTX) Excel (XLSX) Images (PNG, JPG, GIF) Audio (MP3, WAV) HTML XML CSV JSON TXT RTF EPUB ZIP
